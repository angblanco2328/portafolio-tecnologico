🚀 Proyecto Técnico Destacado
Gestión de Cursos – Módulo Matrículas
Descripción:
Aplicación web desarrollada con Java y Spring Boot para gestionar estudiantes, cursos y matrículas.
El módulo de Matrículas permite registrar inscripciones, editar y eliminar registros, y visualizar la información de forma clara y ordenada.
